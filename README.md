# Voxel Fruit Ninja
TDT4230 Final Project - Erlend Ranheim

A real-time voxel renderer implementing a Fruit Ninja-inspired game
using OpenGL 4.3 compute shaders and ray casting.

## Requirements
- CMake 3.15 or higher
- A C++20 compatible compiler
- OpenGL 4.3 compatible GPU and drivers
- Internet connection for first build (CMake fetches GLFW and GLM automatically)

## How to build
1. Open the project folder in an IDE
2. Let CMake configure the project
3. Build and run


## How to play
- Fruits spawn automatically and fly upward under gravity
- Hold left mouse button and swipe across a fruit to slice it
- Fruits can be sliced multiple times

## Controls
- Left mouse button + drag: slice fruit
