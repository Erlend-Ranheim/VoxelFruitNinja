Hi! This is my voxel fruit ninja game that i made for my graphics course in TDT4230.

The project is made using ray pathing and voxels.
The whole project is made from scratch, building a raycaster and voxel rendering with compute shaders for gpu processing for faster rendering. 
THe goal of the project is to mimic the well known fruit ninja game, making a fun and sattisfying little game where you can slice fruit. The voxel models makes it possible to divide the fruit quite cheaply, and its possible to slice remaining voxels multiple times, manipulating how voxel models are built.
