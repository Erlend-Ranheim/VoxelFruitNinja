Voxel Fruit Ninja

Hi!

This is a voxel-based Fruit Ninja-inspired game developed as part of my graphics course (TDT4230).The main goal of the project is to recreate the feel of the well-known Fruit Ninja game by delivering a fun and satisfying slicing experience. By using voxel models, fruits can be split efficiently, and even the remaining pieces can be sliced multiple times. This enables dynamic manipulation of the voxel structures during gameplay.


The project is built entirely from scratch and focuses on real-time rendering using ray casting and voxel-based graphics. Rendering is accelerated using compute shaders on the GPU, allowing for efficient and fast visualization.

Key Features
Custom-built raycaster
Voxel-based rendering system
GPU acceleration with compute shaders
Real-time slicing of voxel objects
